```
Gestion de publicaciones y productos
```

## Description
```
La aplicacion consta de un CRUD de usuarios para publicaciones y productos
```
## Comanndos de nest

```bash
# instalation
$ yarn install
```

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
